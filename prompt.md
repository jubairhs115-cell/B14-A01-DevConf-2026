# AI Prompt Used

Create a unique **Workshop Schedule** section for a **DevConf 2026** conference website.

## Requirements

* Generate **only this section**, not the entire website.
* Use **HTML5 and CSS3 only** (no JavaScript or CSS frameworks).
* Use a `<div>` as the main container.
* Set the main container width to **1479px**.
* Use a **background color of `#EDE9FE`**.
* Add a **2px border with color `#7C3AED`** and rounded corners.
* Include a large centered heading (`<h1>`) and a short subtitle.
* Create a responsive table inside a separate `<div>`.
* Use semantic table elements:

  * `<caption>`
  * `<thead>`
  * `<tbody>`
  * `<tfoot>`
* Include columns for:

  * Workshop
  * Level
  * Duration
  * Speaker
  * Available Seats
* Add at least **six unique workshop entries** related to web development, AI, cloud computing, cybersecurity, DevOps, and UI/UX.
* Use `border-collapse: collapse`.
* Apply alternating row colors and a hover effect.
* Style the table header with a purple theme matching the section.
* Add a footer note indicating that workshop schedules are subject to change.
* Use meaningful, conference-related content instead of placeholder text.
* Keep the design modern, clean, and consistent with the overall DevConf 2026 website.
